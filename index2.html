Assembling Brand Components

Fonts

Oswald-Variable Font

This is used for the first level of headings. Readability is often a concern that is tied to sans-serif fonts, but as it is only used for the heading and is visibility larger than the rest of the elements, readability wouldn't be as much as a concern versus if it had been used for the paragraph elements. we decided to use this font so that it would add more of a professional perspective of the company and ensure it makes the parents feel like they have brought their children to a dependable tutoring center.

Verdana

For larger elements that are not a part of the header, I decided to use a sans-serif Font, Verdana. I thought Verdana would be appropriate because it contrasts well with the serif font used for the header. It does not take the attention away from the company's name, but additionally makes it more playful, allowing the page to be overall, more coordinated with the rest of the content from the page.

Helvetica

Helvetica is used for the elements that require the user to read. I decided to Helvetica after doing some reasearch, and I found out it is considered to be one of the most readable fonts. As a lot of my text is inquiring users of the services available at Auxiliary; I thought from a usability perspective, it would be more approporiate to use this typeface.
